# VUTTR

Projeto com o intuito de disponibilizar uma API e Frontend para gerenciando de links de ferramentas úteis, também conhecido com **VUTTR** *(Very Useful Tools To Remember)*

## Linguagem e Tecnologias

### Backend
- Laravel 8
- PHP 8
- PostgreSQL 13
- Docker (Laravel Sail)
- JWT

### Frontend
- React.JS
- TailwindCSS
- Typescript

## Ferramentas
- VSCode
- API Blueprint
